## running p5 application:

open a terminal (git bash)

navigate to this project
`cd ~/Desktop/creative-coding-2/PROJECT-NAME/`

(use tab after three characters to auto-fill
name, also helps to confirm you're in the right place)

(to see the contents of your current folder, write 'ls')

run the server
`http-server -c-1`

point our browser to localhost:8080

while working on a project:

open a terminal, navigate to your project

do this if you added new files: `git add . `

every time you want to make changes to github:
`git commit -am'commit message goes here'`
`git push`
